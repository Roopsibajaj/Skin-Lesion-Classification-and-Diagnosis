Model Performance Dashboard

Experimental results and metric analysis of the MobileNetV2 CNN classifier

📈 Training Accuracy (Figure 6.2)

Shows the accuracy improvement over 20 epochs. The dashed green line indicates when Phase 2 (fine-tuning) began, unfreezing base layers for fine convergence.

📉 Training Loss (Figure 6.3)

Illustrates the categorical cross-entropy loss reduction. Validation loss stabilizes near 0.18, showing a well-generalized model with no overfitting.

📊 Confusion Matrix (Figure 6.5)

Confusion matrix mapping predicted vs true labels on the test set. Highly populated diagonals show strong classification across all 7 classes.

🏆 Overall Metrics (Figure 6.4)

Summary of standard evaluation metrics: Accuracy (89.2%), Precision (88.7%), Recall (89.2%), and F1-Score (88.9%). All metrics surpass standard clinician baselines.

📊 Dataset Class Distribution

The HAM10000 dataset is heavily unbalanced. Our training pipeline handles this using stratified splitting, minority oversampling, and calculated class weights during training.
